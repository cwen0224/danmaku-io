# 武器系統規格（可直接實作版 v1）

## 1. 目標
- 武器由四維線性屬性決定：`Length(L) / Weight(W) / Center(C) / Sharpness(S)`。
- 支援部位利度：`HeadSharpness`、`ShaftSharpness`。
- 命中後依鋒利區間觸發不同機制：`Bleed`（利）或 `Stun`（鈍）。

## 2. 數值範圍與資料格式
- 所有主屬性範圍：`1.0 ~ 10.0`。
- 武器資料結構：

```ts
interface WeaponData {
  name: string;
  length: number;        // L
  weight: number;        // W
  center: number;        // C
  headSharpness: number; // S(head)
  shaftSharpness: number;// S(shaft)
  baseDamage: number;
  baseKnockback: number;
  baseCooldown: number;
  bleedChanceBase: number;
  bleedDpsBase: number;
  bleedDuration: number;
  baseStunSec: number;
}
```

## 3. 屬性轉換（上限已收斂）

### 3.1 基礎移速
- `MoveSpeed = BaseSpeed * clamp(1 - W / 40, 0.72, 1.0)`

### 3.2 攻擊距離
- `Range = 54 + L * 9`

### 3.3 揮擊頻率（冷卻）
- `Cooldown = clamp(BaseCooldown * (1 - (C - 5.5) * 0.03), 0.18, 0.52)`

### 3.4 傷害核心倍率
- `DamageCore = BaseDamage * (1 + W / 20) * (1 + (C - 5.5) / 15)`

> 設計重點：避免 `C` 導致爆炸倍率，讓武器平衡可控。

## 4. 部位判定與鋒利區間

### 4.1 命中部位
- `hitRatio = clamp((HitDistance - PlayerRadius) / Range, 0, 1)`
- `hitRatio >= 0.68` 視為頭部命中，使用 `headSharpness`。
- 否則使用 `shaftSharpness`。

### 4.2 鋒利區間
- `S >= 7.0`：利刃區（高切割、低擊退、可流血）
- `3.0 < S < 7.0`：混合區（均衡）
- `S <= 3.0`：鈍擊區（高擊退、可僵直）

## 5. 命中公式

### 5.1 傷害
- 初值：`damage = DamageCore`
- 利刃區：`damage *= (1 + (S - 7) * 0.14)`
- 混合區：`damage *= (1 + (S - 5) * 0.05)`
- 鈍擊區：不額外提升傷害

### 5.2 擊退
- 初值：`kb = BaseKB * (1 + W / 12)`
- 利刃區：`kb *= 0.55`
- 混合區：`kb *= 1.0`
- 鈍擊區：`kb *= 1.5 * (1 + (3 - S) * 0.16)`
- 尺寸抵抗：`kb *= 1 / (1 + EnemySizeFactor)`

## 6. 狀態效果

### 6.1 流血（Bleed）
- 觸發條件：`S >= 7.0`
- 機率：`chance = clamp(BleedChanceBase + (S - 7) * 0.09, 0, 0.72)`
- 傷害：`bleedDps = BleedDpsBase * (1 + (S - 7) * 0.18)`
- 持續時間：`BleedDuration`
- 疊加策略：`duration` 與 `dps` 皆取較大值（不做無限堆疊）

### 6.2 僵直（Stun）
- 觸發條件：`S <= 3.0`
- 機率：`chance = clamp(0.22 + EnemySizeFactor * 0.25 + W * 0.02, 0, 0.95)`
- 時間：`stunSec = BaseStunSec * (1 + W / 10) * (1 + EnemySizeFactor * 0.45)`
- 疊加策略：取較長時間覆蓋

## 7. 事件與 VFX / SFX
- 命中事件：`onHit(enemyId, hitType, damage, knockback, sharpness, hitRatio)`
- 利刃命中：Slash VFX + 清脆音效
- 鈍擊命中：Impact VFX + 低頻音效 + 短震屏

## 8. 實作優先順序
1. 建立 `WeaponData` 與 runtime 派生參數。
2. 實作 `calculateHitEffect()`（含部位判定）。
3. 接上 `Bleed/Stun` 的更新循環。
4. 補上事件匯流排與 VFX/SFX 鉤子。
5. 最後做平衡表（武器、敵人尺寸、狀態上限）。
